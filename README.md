# 🤖 Telegram Payment Bot

Telegram-бот на Python с поддержкой встроенной системы платежей Telegram.

Бот демонстрирует процесс оплаты: пользователь может отправить команду `/pay`, после чего бот выставит счет. После успешной оплаты бот отправляет подтверждение.

---

## ⚙️ Возможности

Бот умеет:

* отправлять счет для оплаты
* обрабатывать подтверждение оплаты
* отправлять сообщение после успешного платежа
* использовать переменные окружения для безопасности токенов

---

## 🛠 Используемые технологии

* Python
* pyTelegramBotAPI
* Telegram Payments API
* python-dotenv

---

## 📦 Установка

Склонируйте репозиторий:

git clone (https://github.com/IvanVIAB/telegram-paid-bot)

Перейдите в папку проекта:

cd telegram-payment-bot

Установите зависимости:

pip install -r requirements.txt

---

## 🔑 Настройка

Создайте файл `.env` в корне проекта.

Добавьте туда токены:

TELEGRAM_TOKEN=your_telegram_bot_token
PROVIDER_TOKEN=your_payment_provider_token

Где:

* **TELEGRAM_TOKEN** — токен вашего Telegram-бота
* **PROVIDER_TOKEN** — токен платежного провайдера Telegram

---

## ▶️ Запуск

Запустите бота командой:

python bot.py

После запуска бот будет ожидать команды пользователей.

---

## 💳 Как работает оплата

1. Пользователь отправляет команду `/start`
2. Бот предлагает протестировать оплату
3. Пользователь вводит `/pay`
4. Бот отправляет счет
5. После успешной оплаты бот отправляет сообщение с подтверждением
